# nubia_z80ultra_gcam
ZTE Nubia Z80 Ultra GCam mod configs
Phone: ZTE Nubia Z80 Ultra
Firmware: MyOS16.0.16_NX741J_EEA
Main camera sensor: OmniVision OV50H 1/1.28" +DCG
Wide camera sensor: OmniVision OV50E 1/1.55"
Tele camera sensor: OmniVision OV64B 1/2.0" +ISZ

LMC GCam
APK: LMC9.6 Release 1 Fix2 Samsung.apk
Config: z80u_DCG_ISZ_byVT.xml
Features:
Main camera with DCG option (less noise in shadows)
Tele camera with ISZ option (zoom to 140mm - sensor crop without binnung)

описание: Настроены все задние камеры для режимов день и полумрак на баланс резкости и гладкости с меньшим акцентом на резкость, чем у AGC и с большей гладкостью.
Резкость чуть меньше, чем у AGC конфига при попиксельном сравнении, в основном на телевике.
В отличие от AGC здесь для некоторых камер активирована недоэкспозиция кадра с последующей подтяжкой теней в обработке - лучше при съемке против света, но чуть больше шума в тенях.
